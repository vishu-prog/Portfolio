My Web Development Journey

🌱 Getting Started (Month 1)

Started learning HTML & CSS through freeCodeCamp and YouTube tutorials. Built my first static webpage — a personal "About Me" site with basic styles.

📐 Layout & Flexbox (Month 2)

Learned how to use Flexbox and CSS Grid to create responsive layouts. Practiced by recreating website headers and footers.

🎨 Tailwind CSS & Design (Month 3)

Discovered Tailwind CSS and built several landing pages. Started focusing on clean design, spacing, and UI components.

🧠 JavaScript Fundamentals (Month 4–5)

Learned the basics of JavaScript — variables, loops, functions, arrays, DOM manipulation. Built a to-do app and simple calculator.

🔗 APIs & Projects (Month 6)

Learned to fetch data from APIs using JavaScript. Created a Weather App using OpenWeather API and a Blog Reader using a JSON API.

⚛️ Learning React (Month 7)

Started learning React: components, props, state, and hooks. Rebuilt my portfolio using React + Tailwind. Deployed my first React project.

🚀 Ongoing Growth (Now)

Working on full-stack basics (Node.js, MongoDB). Learning Git, hosting, and continuing to build real-world projects. Writing blog posts to share my progress!
